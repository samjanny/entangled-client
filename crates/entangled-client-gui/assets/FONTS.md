# Embedded fonts

This directory bundles the DejaVu fonts, embedded into the binary for consistent
typography (and real bold/monospace) regardless of the host's installed fonts:

- `DejaVuSans.ttf` - proportional body font
- `DejaVuSans-Bold.ttf` - bold weight
- `DejaVuSansMono.ttf` - monospace (code)
